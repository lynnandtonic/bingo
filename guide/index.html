BINGO Call Board Guide

If you are hosting a BINGO party, put this Call Board up on a TV or monitor so players can see which numbers have been called.

This guide assumes you know how to play BINGO, but if you need a refresher here’s wikiHow’s How to Play Bingo.

This guide also assumes you are using a physical BINGO ball cage to draw numbers. If you are looking for a fully digital version, I suggest using letsplaybingo.io.

Pro tip: One thing I like to start with—even for those who have played before—is to point out that no number will be in more than one letter column. B column is numbers 1-15, I column is numbers 16-30, and so on. Something most people haven’t thought about!

Colors

If you are using BINGO paper sheets like this:

You may want to match the Call Board color with the color BINGO sheet the players should be using. It helps make sure everyone is on the right sheet. Click a color to swap.

Numbers

As you draw numbers, you can keep track of them on the Call Board. Click each number and it will get highlighted. The current number will automatically be displayed large in the middle.

As you call numbers, players might miss one, so there is a smaller “previous number” for the caller’s information in case anyone asks.

All your drawn numbers will be highlighted (make sure to click them!), so they can also check against those.

Clearing

In between games, you can quickly clear the number board by clicking the Clear button in the bottom right.

If you’d like to reset the Call Board to its original state (including color and example pattern), you can click the Reset button in the bottom left.
